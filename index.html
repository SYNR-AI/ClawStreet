ClawCapital

Your AI bot, promoted to portfolio manager. It watches the market 24/7, explains every trade, and never takes a vacation.

We were itching to let our bot trade real money, but decided to let it paper-trade first and prove itself. So here we are—a $1M simulated portfolio, live market data, and an AI that has to justify every move.

Fork of OpenClaw · main syncs upstream · features on clawcapital

What your "PM" can do

Trade — long, short, everything

  • Long & Short with isolated margin (1x)
  • US stocks + crypto in one portfolio
  • Position Stories — thesis, memo, trade history
  • Day P&L tracking with daily snapshots

Watch — like a hawk, but smarter

  • The Eye — per-ticker AI monitoring focus
  • Heat Rating — cold / warm / hot
  • Intel Feed — AI-pushed news with impact
  • Opportunity Radar — cross-sector scanning

Dashboard

  • Portfolio — P&L, margin, expandable stories
  • Watchlist — intel board (Stocks / Crypto)

Live Market Data

  • Binance REST API for crypto
  • Yahoo Finance for US stocks
  • 30s cache, auto-refresh

Tools the bot uses on its own

Tool What it does
portfolio_buy / sell / short / cover Execute trades
portfolio_get Check holdings, cash, P&L, margin
portfolio_set_meta Record thesis & notes on positions
market_data_quote Get live prices (crypto & stocks)
watchlist_add / remove / update Manage watchlist
watchlist_add_intel Push intelligence to the feed

Quick start

# clone & switch to the clawcapital branch
git clone <this-repo>
cd ClawCapital && git checkout clawcapital

# build
pnpm install && pnpm ui:build && pnpm build

# run
pnpm openclaw onboard --install-daemon
pnpm openclaw gateway --port 18789

Architecture

Gateway (ws://127.0.0.1:18789)
  ├── src/portfolio/          # Trading engine, market data, margin
  ├── src/watchlist/          # Watchlist domain + enrichment
  ├── src/agents/tools/       # portfolio-tools, watchlist-tools
  ├── src/gateway/server-methods/
  │     ├── portfolio.ts      # RPC: get, trade, quote, reset
  │     └── watchlist.ts      # RPC: get, add, remove, update
  └── ui/src/ui/views/        # Portfolio & Watchlist pages
